# 🧠 Alex - AI Personal Tutor

> A personal AI tutor that actually teaches you through 
> real conversation — not lectures, not walls of text.

---

## 🎯 The Problem

Most study tools give you information and leave.

They dump explanations, roadmaps and quizzes and say:
**"Go figure it out yourself."**

That's not teaching. That's a textbook.

---

## ✅ The Solution

Alex is different.

Alex teaches like a real human tutor:
- Asks what you already know
- Teaches one concept at a time
- Checks if you understood
- Adapts if you're confused
- Never moves on until you're ready
- Celebrates your progress

---

## ✨ Features

- 🧠 **Adaptive Learning** - Adjusts to your knowledge level
- 💬 **Real Conversation** - Back and forth like texting a tutor
- 📝 **Understanding Checks** - Never assumes you understood
- 🔄 **Never Gives Up** - Tries different explanations if confused
- 🏆 **Progress Celebration** - Encourages you specifically
- 🌙 **Dark Mode UI** - Clean and professional interface
- 📱 **Any Topic** - Python, Math, History, Science, anything

## 🚀 How To Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/pablo54564/ai-personal-tutor.git
cd ai-personal-tutor
```

### 2. Install requirements
```bash
pip install streamlit groq python-dotenv
```

### 3. Create .env file
```
GROQ_API_KEY=your-groq-key-here
```

Get your free Groq API key at: https://console.groq.com

### 4. Run the app
```bash
python -m streamlit run tutor.py
```

### 5. Open browser
```
http://localhost:8501
```

## 💬 How It Works
You type a topic
↓
Alex asks your current level
↓
Alex teaches first concept only
↓
Alex checks your understanding
↓
You reply
↓
Alex adapts based on your answer
↓
Moves to next concept only when ready
↓
Gives mini challenges
↓
Celebrates your progress
